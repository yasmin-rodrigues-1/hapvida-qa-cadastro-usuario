# Teste QA Manual — Cadastro de Usuário

O repositório contém a entrega da prova técnica para a posição de QA, com:

- Plano de teste resumido (escopo, tipos de testes e estratégia de execução)
- Casos de teste em Gherkin para o módulo de Cadastro de Usuário (nome, e-mail e senha)

## Estrutura

- `docs/plano_de_testes.md` → Plano de teste resumido
- `features/cadastro_usuario.feature` → Casos de teste em Gherkin

## Escopo do objeto de teste

Tela de Cadastro de Usuário com campos:
- Nome
- E-mail
- Senha
